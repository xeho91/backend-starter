# Back-end project template

## Project structure

```text
[0] .
├── [1] .husky
├── [2] apps
├── [3] packages
└── [4] types
```

Where as:

-   `0` -> root of the project
-   `1` -> Git hooks used with [husky](https://github.com/typicode/husky)
-   `2` -> binary packages _(applications)_
-   `3` -> library packages
-   `4` -> types packages
